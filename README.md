# 📍 Around The U.S.

"Around The U.S." es una página web interactiva que funciona como una galería de fotos personal. Permite a los usuarios editar su perfil y añadir o eliminar imágenes de sus lugares favoritos.

Este proyecto fue desarrollado para aplicar conceptos de JavaScript, manipulando el DOM para dar vida a una maqueta estática y crear una experiencia de usuario dinámica.

## ✨ Funcionalidades

*   **📝 Edición de perfil:** Modifica el nombre y la descripción del usuario.
*   **🖼️ Añadir nuevas tarjetas:** Agrega nuevos lugares a la galería desde un formulario.
*   **🗑️ Eliminar tarjetas:** Cada tarjeta cuenta con un botón para ser eliminada.
*   **❤️ "Me gusta":** Marca tus tarjetas favoritas haciendo clic en el icono del corazón.
*   **🔍 Vista previa de imágenes:** Haz clic en una imagen para verla en un tamaño mayor.
